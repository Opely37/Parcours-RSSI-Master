🔍 Investiguez un incident de sécurité

Analyse forensique complète d'une compromission système par PDF malveillant. Reconstitution de la chaîne d'attaque en 36 secondes, extraction de tous les IOC, cartographie MITRE ATT&CK et plan de remédiation priorisé. Classifié CRITIQUE — Remote Code Execution.

  • Analyse mémoire RAM (4 GB) avec Volatility 3 — extraction processus, connexions réseau, lignes de commande
  • Analyse image disque E01 (10,6 GB) avec Sleuthkit — récupération des fichiers malveillants par inode
  • Reverse engineering du PDF avec pdf-parser — JavaScript malveillant exportDataObject() / nLaunch:2
  • Analyse macro VBA AutoOpen() avec oletools — détection Shell() dans verybad.doc
  • Identification chaîne anormale : AcroRd32.exe → WINWORD.EXE → cmd.exe → ntvdm.exe
  • Serveur C&C identifié : 2.16.117.17 (Akamai) — communication HTTP non chiffrée
  • 6 techniques MITRE ATT&CK cartographiées (T1566, T1059, T1055, T1027, T1071, T1218)
